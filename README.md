## Getting Started

To get a local copy up and running follow these simple example steps.

`git clone https://github.com/eng-mohamed-eldeeb/protfolio-v2.0.git`

Then `cd protfolio-v2.0`

Then `npm install` to install packages

Finally `npm start`
